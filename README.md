# NekoHome
一个使用 React 的导航页
## 使用

安装依赖：

```shell
npm install
```

## 配置

配置文件为 `/config.ts` ，可以照着注释修改

## 编译

```shell
npm run build
```

生成的文件存放在 `/dist` 目录下，将其上传至静态空间即可
# 留言板后端部署
首先进入到/server文件夹

安装依赖：

```shell
npm install
```

运行：

```shell
node index.js
```

服务端将会开启在http://localhost:7799
